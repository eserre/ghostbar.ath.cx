--- layout: post title: Help XScreenSaver on Debian! tags: - Debian - English - Planet Debian - Software - XScreenSaver status: publish type: post published: true meta: _edit_last: '1' dsq_thread_id: '123511113' --- XScreenSaver is of the most widely used screensavers app in Linux systems and works as well as a huge collection of screensavers for another apps like kscreensavers. In Debian is maintained by only 2 persons which has in this moment lack of time. Because of this we're looking for co-maintainers that should accomplish the following main tasks:
  • Debugging actual tickets and check which ones exists yet.
  • Fixing bugs, creating and applying patches.
  • Check any issue that new upstream-releases may give to packages depending on xscreensaver like kscreensaver.
The package is hosted over git repositories, the official repo is at git.debian.org under collab-maint/xscreensaver.git and there's a copy under my GitHub account in deb-xscreensaver so if you're interested you should learn something about maintaining debian packages with git and this guide of git-buildpackage could be useful for that. There are some tasks that needs to be done before releasing 5.08 (which is basically done):
  • Finishing debian/copyright that was practically empty leaving away many hacks and their proper copyright statement.
  • Checking which bugs were closed by the last upstream-release.
  • Checking with the KDE team which issues may give the new release with kscreensaver (which happens very often).
There's a pending approval for a project under alioth for maintaining XScreenSaver as a group because is a package that requires a lot of time and a group is the ideal way to work with this. If you're interested answer to bug #521783, make a fork of the git repo and start working! We need all the help you can give!
